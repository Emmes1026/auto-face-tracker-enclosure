# face-gesture-tracker-enclosure
3D printable enclosure for an automatic face tracking DIY project.

# Project Overview
This repository contains the 3D model files (designed in FreeCAD) for the enclosure of my other project **Automatic Face Tracking with Hand Gesture Recognition**.

**Note: [The software and logic for the main tracking system are separate.] (https://github.com/Emmes1026/face-gesture-tracker/tree/main)**

# Description

This model consists of 3 parts:

* Main Enclosure: Features two separate chambers. The front chamber is designed to hold the Raspberry Pi, while the rear chamber houses the PCA9685 module.
* Mounting Bracket: A custom clamp to hold the original Raspberry Pi case firmly in place.
* Rear Cover: A protective flap to enclose and hide the PCA9685 board in the back chamber.

Note: All dimensions were designed around the official Raspberry Pi 5 case.

# Components List

To build the fully functional setup, the following parts were used:

* Raspberry Pi 5 with the official case
* PCA9685 (servo driver)
* 3D-printed enclosure
* 2x SG90 micro servos
* Unofficial Camera Module v2 (by SHCHV)
* Pan and Tilt camera mount from Thingiverse: [Thingiverse Project Link](https://www.thingiverse.com/thing:4710301)
